1. Pull latest version of application using Github Desktop (in top top left)
2. go to apis/feedurls.txt and add any URLs, make sure that they are formatted the same as previous
3. Save file
4. Back on github desktop, you should see changes that have been made by yourself. If anything changed that is not feedurls, unselect it's respective checkbox.
5. Once done, write a small commit message in bottom left of application
6. Select commit in in bottom left.
7. Once commited, there should be an option to push the changes where the pull button originally was. Press this to finalise the changes.

P.S. Once changes are commited they will affect the public site, so do not push unwarrented changes.
